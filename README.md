Penjelasan singkat tentang fungsi dan cara kerja program:
- Program ini menggunakan scanner untuk input data dari pengguna.
- Variabel jumlahSiswa menyimpan banyaknya siswa, dan totalNilai menjumlah semua nilai.
- Perulangan for digunakan untuk meminta nilai setiap siswa lalu menambahkannya ke totalNilai.
- Setelah itu, rata-rata dihitung dengan totalNilai / jumlahSiswa dan hasilnya ditampilkan.
